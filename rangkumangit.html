Intro

pada materi ini menjelaskan dasar-dasar git dan menjelaskan beberapa tools yang digunakan dalam menajalankan git.

Berikut tools yang digunakan

  • GIT -> digunakan untuk control dalam pembangunan project
  • Akun GITLAB/GITHUB -> layanan yang digunakan untuk project git bersama
  • Sourcetree -> untuk menggunakan git selain cli
  • VsCode -> sebagai text editor
  • Lightshoot -> digunakan untuk mengcapture layar
  • Grepper -> berguna untuk memberi pintasan dari hasil browser yang dicari
  • Screen O Matic -> untuk merekam layar
  • Zoom / Gmeet -> untuk melakukan pertemuan atau diskusi

    • Fundamental

    pada materi ini menjelaskan perintah-perintah dasar dalam menjalankan git

    Berikut materi yang dibahas

  • Git init -> untuk menginisialisai repository
  • Git config -> untuk mengkonfigurasi akun email dan username
  • Git status -> untuk mengetahui status file
  • Git add . -> untuk memasukkan file ke area staging
  • Git commit -> untuk mencatat perubahan yang telah dibuat
  • Git log -> untuk melihat riwayat commite
  • Git clone -> untuk mengkopi repository dari server
  • Git remote -> untuk berinteraksi antara client dan server
  • Git push -> untuk mengirim commite ke server
  • Git pull -> untuk mengambil commite dari server

    • Branch

    pada materi ini menjelaskan tentang percabangan dari git, yaitu untuk pengerjaan suatu project secara tim

    Berikut materi yang dibahas

  • Git branch -> untuk melihat semua branch (local)
  • Git branch --all -> untuk melihat semua branch (public)
  • Git branch Nama_branch -> untuk membuat branch baru
  • Git checkout Nama_branch -> untuk masuk kedalam branch lainnya
  • Git branch -d Nama_branch -> untuk menghapus branch
  • Git merge Nama_branch -> untuk menggabungkan commite dari dua branch

    • GUI Tools

    pada materi ini belajar cara menggunakan sourcetree, yaitu suatu tools yang berguna untuk menggunakan git tanpa menggunakan cli di comandprompt,namun menggunakan sebuah software berbasis GUI
